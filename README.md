# clipboard.aux2

ファイル・画像・テキストをクリップボードから直接貼り付けるAviUtl2プラグイン。

## インストール

[Releases](https://github.com/sevenc-nanashi/clipboard.auf2/releases/latest)から`clipboard-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
